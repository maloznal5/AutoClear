# DataBridge Pro | Enterprise Data Automation

Автономный ETL-движок для синхронизации данных между внешними API и бизнес-системами.

## Технические возможности
- **Connectors:** Модульная система для интеграции с REST API, SQL и NoSQL.
- **Transformation:** Потоковая обработка и валидация данных перед импортом.
- **Automation:** Полная поддержка запуска через CRON на Linux/Termux.

## Стек
- Python 3.12, SQLite3, JSON-Schema, Bash.
